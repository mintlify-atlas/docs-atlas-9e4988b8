## GET /api/provinces/

Returns a list of all provinces in Nepal with their administrative information including area, website, and headquarters.

### Response

Returns an array of province objects.

<ResponseField name="id" type="integer" required>
  Province ID
</ResponseField>

<ResponseField name="name" type="string" required>
  Province name
</ResponseField>

<ResponseField name="area_sq_km" type="string" required>
  Area in square kilometers
</ResponseField>

<ResponseField name="website" type="string" required>
  Province website URL
</ResponseField>

<ResponseField name="headquarter" type="string" required>
  Province headquarter city
</ResponseField>
